## 🚀 Usage (Inference)
This model is compatible with standard inference backends like vLLM.

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model_id = "smolify/smolified-financial-statement-normalizer"
tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(model_id, device_map="auto")

messages = [
    {"role": "system", "content": '''You are a technical lead at a Fintech firm.'''},
    {"role": "user", "content": '''What is the key advantage of an SLM over a cloud-hosted LLM when dealing with confidential financial datasets inside local server clusters?'''}
]
text = tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True).removeprefix('<bos>')

from transformers import TextStreamer
_ = model.generate(**tokenizer(text, return_tensors="pt").to("cuda"), max_new_tokens=1000, streamer=TextStreamer(tokenizer, skip_prompt=True))
```
